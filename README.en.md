## 💡 About the Project

**Awesome Kanban Board** is a lightweight, interactive task management app  
that demonstrates clean React architecture and client-side data persistence.

The project highlights component-based structure, reusable logic,  
and minimalist design with no external UI libraries.

---

## ⚙️ Tech Stack

⚛️ React 19 — современный JavaScript-фреймворк
🧭 React Router DOM 7 — маршрутизация и навигация
🏗️ Create React App — базовая среда разработки и сборка
💾 LocalStorage API — хранение данных в браузере
🎨 CSS-модули / inline-стили — оформление интерфейса
🧠 JavaScript (ES6+) — логика компонентов и состояние
🌐 GitHub Pages — деплой и хостинг
🧩 Node.js + npm — зависимости и сборка

## 🚀 Features

- Add new tasks to **Backlog**
- Move tasks through stages: _Ready → In Progress → Finished_
- Track counts of active and finished tasks
- Automatically save board state using `localStorage`
- Reset all data with the **Reset Board** button
- Clean structure and intuitive logic
